## Interpretação funcional

Esta tela concentra o fluxo de criação, carga e edição de um programa de inspeção:

1. criar ou carregar programa
2. selecionar câmera
3. mover sistema para posição desejada
4. capturar imagem
5. definir região da placa
6. desenhar ROIs
7. salvar/concluir
