# Ação: Carregar Programa

## Tela

[[tela_projeto|Tela Projeto]]

## Ação do usuário

Usuário clica em **Carregar Programa**.

## Ação esperada do sistema

- abrir seletor de programa existente
- carregar dados do programa
- preencher contexto da tela
- restaurar imagens, ROIs e posições relacionadas

## Pontos a confirmar

- se carrega programa completo ou só câmera atual
- se carrega também posições de captura
