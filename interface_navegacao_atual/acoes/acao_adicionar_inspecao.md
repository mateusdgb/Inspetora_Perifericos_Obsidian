# Ação: Adicionar Inspeção

## Tela

[[tela_projeto|Tela Projeto]] → subaba **Inspeção**

## Ação do usuário

Usuário clica em **Adicionar Inspeção**.

## Ação esperada do sistema

- criar uma nova configuração/regra de inspeção no contexto atual
- permitir associar parâmetros de threshold, polaridade, percentual e similaridade
- adicionar o item à estrutura do programa

## Pontos em aberto

- se a nova inspeção é vinculada a uma ROI existente
- se exige seleção prévia de imagem/região/item
