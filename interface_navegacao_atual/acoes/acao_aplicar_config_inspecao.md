# Ação: Aplicar Configuração de Inspeção

## Tela

[[tela_projeto|Tela Projeto]] → subaba **Inspeção**

## Ação do usuário

Usuário clica em **Aplicar Config**.

## Ação esperada do sistema

- validar os parâmetros preenchidos
- aplicar os parâmetros ao contexto atual de inspeção
- refletir visualmente ou internamente a nova configuração
- deixar a configuração pronta para salvamento do programa

## Pontos em aberto

- se a aplicação é imediata
- se atua em memória temporária
- se altera uma inspeção já selecionada ou cria nova
