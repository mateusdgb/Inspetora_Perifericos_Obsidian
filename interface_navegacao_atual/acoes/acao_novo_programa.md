# Ação: Novo Programa  
  
## Tela  
  
[[tela_projeto|Tela Projeto]]  
  
## Ação do usuário  
  
Usuário clica no botão **Novo Programa**.  
  
## Ação esperada do sistema  
  
- abrir fluxo de criação de programa  
- solicitar nome  
- validar duplicidade  
- criar estrutura do programa  
- preparar contexto para captura e edição  
  
## Dependências prováveis  
  
- program manager  
- persistência de projeto  
- inicialização de estrutura por câmera
