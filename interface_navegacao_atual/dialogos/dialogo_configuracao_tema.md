# Diálogo: Configuração de Tema

## Objetivo

Esta configuração define a aparência visual do sistema.

Pelos prints, os modos disponíveis são:
- Auto (detectar do sistema)
- Claro
- Escuro

---

## Significado das opções

### Auto
O sistema tenta seguir o tema do sistema operacional.

### Claro
A interface usa visual claro.

### Escuro
A interface usa visual escuro.

---

## Importância

O tema não afeta a lógica de negócio, mas impacta:
- usabilidade
- conforto visual
- contraste
- leitura em ambiente industrial
